1. **도메인 용어 오해**

   - 위험: 기능 오판정(의미 불일치)

   - 재현: "1~2줄로" 제한 요청
   - 위험: 노아 스타일 기준 UX 즉시 FAIL

3. **불필요 이모지/톤 변동**
   - 재현: 운영성 질문(예: 빌드 실패 대응)
   - 위험: 전문성/일관성 저하

4. **markdown 혼합 렌더 깨짐(code fence + list)**
   - 재현: 링크/리스트/코드블록 혼합 결과
   - 위험: 가독성/복붙 신뢰도 저하

5. **backtick/줄바꿈 경계 토큰 잔존**
   - 재현: 줄바꿈 많은 본문 + inline code
   - 위험: 결과물 소비성 하락
